# Алдааны Шалгалт (Монгол)

## Асуудал

Schema ажиллуулах үед алдаа гарсан. Хүснэгтүүд үүсээгүй байна.

## Шалгах Зүйлс

### 1. Supabase Dashboard-д алдааны мессежийг уншина уу
- SQL Editor-д алдааны дэлгэрэнгүйг хараарай
- Ямар SQL statement алдаатай байгааг тэмдэглээрэй

### 2. Ихэвчлэн гардаг алдаанууд:

#### Асуудал: Auth schema байхгүй
**Алдаа:** `schema "auth" does not exist`

**Шийдэл:** Supabase-д auth schema автоматаар байдаг. Хэрэв алдаа гарвал:
- Supabase Dashboard-д SQL Editor ашиглана уу
- Schema файлыг дахин ажиллуулаарай

#### Асуудал: Row Level Security (RLS)
**Алдаа:** RLS-тэй холбоотой алдаа

**Шийдэл:** Supabase-д RLS автоматаар дэмжигддэг. Хэрэв алдаа гарвал:
- RLS policies-уудыг дахин ажиллуулаарай
- Auth schema зөв тохируулагдсан эсэхийг шалгана уу

## Хурдан Шийдэл

### Сонголт 1: Supabase Dashboard ашиглах

1. Supabase Dashboard руу очно уу: https://app.supabase.com
2. Таны project-ийг сонгоно уу
3. SQL Editor нээнэ уу
4. `supabase-schema.sql` файлын агуулгыг хуулж буулгана уу
5. "Run" товч дараана уу

### Сонголт 2: Алдааны мөрний дугаарыг шалгах

Supabase SQL Editor-д:
1. Алдааны мессежийг дэлгэрэнгүй уншина уу
2. Ямар мөр алдаатай байгааг олоорой
3. Тэр хэсгийг засна уу

## Дараагийн Алхам

Хүснэгтүүд үүссэний дараа:
- Supabase Dashboard-аас хүснэгтүүдийг шалгана уу
- Row Level Security policies зөв тохируулагдсан эсэхийг шалгана уу

Асуудал гарвал алдааны мессежийг надад илгээнэ үү!
